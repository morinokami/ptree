# ptree

ptree prettifies `tree` command with emojis like this:

```
$ ptree              
📁 .
├── 🤖 binary.bin
├── 📄 document.txt
├── 🦏 javascript.js
├── 📝 markdown.md
├── 🐪 perl.pl
├── 🐍 python.py
└── 🦀 rust.rs

0 directory, 7 files
```

## Installation

```sh
$ npm install -g ptree
```
